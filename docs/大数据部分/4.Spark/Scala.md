# Scala

## 概述

版本问题

+ 目前 1.6版本是比较稳定的，很多公司也在用
+ spark2.X使用的是scala 2.x

六大特性

+ 运行在Java虚拟机，可以使用Java的包
+ 类型可变，自动推断，介于J和Java之间
+ 支持并行和分布式
+ 抽像类，定义方法，类似java的抽线类和接口的整合
+ switch可进行多类型匹配

## 安装Scala

+ 安装环境变量

## 语言特性

### 基本数据类型

+ Unit 表示无返回， == viod
+ null 空引用
+ Nothiing 表示没有值，是所有类的子类
+ Any 虽有类型的超类
+ AnyRef所有引用类型的超类
+ AnyVal 所有值的超类
